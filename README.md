# 每日60s新闻插件 for Typecho

> 提供「每天60秒读懂世界」新闻早报功能，支持短代码 `[daily60s]` 嵌入任意文章或页面。

## 功能特性

* **短代码嵌入**：在任意文章或页面中使用 `[daily60s]` 即可显示每日新闻

* **多 API 备用切换**：主 API 失败时自动尝试备用 API，保证稳定性

* **API Token 支持**：兼容需要 token 的 API（如 ALAPI）

* **多种返回格式**：支持 JSON（文本+图片）、仅文本、图片三种模式

* **图片本地化**：JSON 模式下可将早报图片下载到本地，避免外链失效

* **懒加载**：图片滚动到可视区域才加载，提升页面性能

* **SEO 结构化数据**：自动输出 NewsArticle 类型的 JSON-LD 数据

* **缓存并发锁**：防止高并发下重复请求 API

* **定时自动预热**：配合宝塔计划任务，每天凌晨自动获取新闻

* **缓存管理面板**：后台可视化管理缓存文件，支持刷新、清空、删除

* **自定义外观**：标题文案、副标题、标题栏渐变配色可自定义

## 环境要求

* Typecho 1.3.0 及以上版本

* PHP 7.4+（推荐 PHP 8.0+）

* `usr/uploads/60s/cache/` 目录需可写（插件会自动创建）

## 安装方法

### 方法一：直接下载

1. 下载本仓库的所有文件

2. 将 `Daily60s` 文件夹上传到 Typecho 的 `usr/plugins/` 目录

3. 登录 Typecho 后台 → 控制台 → 插件 → 启用「每日60s新闻」

### 方法二：Git 克隆

```bash
cd usr/plugins/
git clone https://github.com/227qidian/Daily60s.git Daily60s
```

然后登录后台启用插件。

## 使用方法

### 基本使用

在任意文章或页面的正文中输入短代码：

```
[daily60s]
```

保存后访问该页面即可显示每日60s新闻。首次访问时自动从 API 获取当天新闻并缓存，之后全天读取缓存快速响应。

### 定时自动预热（可选）

如需每天凌晨自动获取新闻（用户访问时直接读缓存），可配置定时任务：

1. 在插件设置中开启「定时自动预热」

2. 登录宝塔面板 → 计划任务 → 添加任务

3. 任务类型选择「访问URL」

4. 执行周期设为每天，时间建议设为 06:00

5. URL 填写：

```
https://你的域名/action/daily60s?do=preheat
```

或使用 Shell 命令：

```bash
curl -s "https://你的域名/action/daily60s?do=preheat"
```

## 配置项说明

### API 设置

| 配置项       | 说明                                                        |
| --------- | --------------------------------------------------------- |
| 主 API 地址  | 默认 `https://60s.viki.moe/v2/60s?encoding=json`，免费无需 token |
| 备用 API 地址 | 每行一个，主 API 失败时按顺序尝试                                       |
| API Token | 部分 API（如 ALAPI）需要，支持 `{token}` 占位符替换                      |
| API 返回类型  | JSON 格式 / 仅文本格式 / 图片格式                                    |

### 显示设置

| 配置项      | 说明                            |
| -------- | ----------------------------- |
| 自定义标题文案  | 默认「每天60秒读懂世界」                 |
| 自定义副标题   | 显示在标题下方，留空则不显示                |
| 标题栏渐变起始色 | 默认 `#667eea`                  |
| 标题栏渐变结束色 | 默认 `#764ba2`                  |
| 显示样式     | 完整显示 / 仅图片 / 仅文字（仅 JSON 模式有效） |
| 显示日期标题   | 是否显示标题和日期                     |

### 图片设置

| 配置项   | 说明                                         |
| ----- | ------------------------------------------ |
| 图片本地化 | JSON 模式下将图片下载到本地，避免外链失效                    |
| 图片懒加载 | 使用 `loading="lazy"` + IntersectionObserver |

### SEO 设置

| 配置项   | 说明                                     |
| ----- | -------------------------------------- |
| 结构化数据 | 输出 NewsArticle 类型的 JSON-LD 数据，利于搜索引擎收录 |

### 缓存设置

| 配置项    | 说明                             |
| ------ | ------------------------------ |
| 缓存保留天数 | 默认 15 天，超过此天数的缓存自动清理，设为 0 则不清理 |

## 缓存管理

插件提供后台缓存管理面板：控制台 → 每日60s缓存管理

支持以下操作：

* **刷新缓存**：清空所有缓存并重新获取当天数据

* **清空所有缓存**：仅清空，不重新获取

* **删除单个文件**：删除指定的缓存文件

* **查看缓存统计**：查看缓存文件数量、总大小等

## 缓存目录

所有缓存文件统一存放在：

```
usr/uploads/60s/cache/
```

目录不存在时会自动创建。如遇 404 错误，通常是目录权限问题，请执行：

```bash
chown -R www:www usr/uploads/60s/
chmod -R 755 usr/uploads/60s/
```

## Action 接口

插件注册了以下 Action 接口（需管理员权限，`preheat` 除外）：

| 接口                                                | 说明                     |
| ------------------------------------------------- | ---------------------- |
| `/action/daily60s?do=preheat`                     | 定时预热（无需 token，供计划任务调用） |
| `/action/daily60s?do=refresh&_redirect=1`         | 刷新缓存（需 token）          |
| `/action/daily60s?do=clear&_redirect=1`           | 清空缓存（需 token）          |
| `/action/daily60s?do=delete&file=xxx&_redirect=1` | 删除单个文件（需 token）        |
| `/action/daily60s?do=stats`                       | 获取缓存统计（需 token）        |

## 常见问题

### 新闻或图片加载不出来（404）

**原因**：缓存目录无法自动创建，通常是 `usr/uploads/60s/` 目录所有者不是 Web 服务器用户。

**解决**：在宝塔面板中将 `usr/uploads/60s/` 目录所有者设为 `www`，权限设为 `755`。或 SSH 执行：

```bash
chown -R www:www usr/uploads/60s/
chmod -R 755 usr/uploads/60s/
```

### API 请求失败

* 检查主 API 地址是否正确

* 配置备用 API 地址（每行一个）

* 如使用需要 token 的 API，请正确填写 API Token

* 检查服务器是否允许外联（`curl`、`file_get_contents`）

### 缓存不更新

* 在缓存管理面板点击「刷新缓存」

* 检查缓存保留天数设置

* 清空缓存后重新访问页面

## 默认 API

| API      | 地址                                           | 说明             |
| -------- | -------------------------------------------- | -------------- |
| JSON API | `https://60s.viki.moe/v2/60s?encoding=json`  | 返回 JSON 数据（默认） |
| 图片 API   | `https://60s.viki.moe/v2/60s?encoding=image` | 直接返回图片         |
